Install

Brief Setup Instructions: First you need to add the cache for your client to the appropriate folder. For 317, add the cache to apollo/data/fs/317. For 377, add the cache to apollo/data/fs/377. You can build Apollo with Apache Ant and Apache Ivy. First download and setup Ant and Ivy, info on that can be found here: Apache Ant - Welcome and here: Home | Apache Ivy â„¢. In command prompt, in the apollo directory, run these commands: ant resolve "ant resolve" only needs to be called once when you first download Apollo and again when you update the versions of the dependencies. It downloads third-party libraries such as JRuby and Netty. ant build "ant build" compiles all the source files. Code: ant run "ant run" starts the server running, by default it tries to load the 377 cache and packet codecs. You can change this by editing the version attribute in the build.xml file, or by running it manually without ant. Other ant targets you may find useful: doc - generates javadocs, if you have graphviz installed it also includes nice graphs of the class hierarchy test - runs (the very limited set of) unit tests
